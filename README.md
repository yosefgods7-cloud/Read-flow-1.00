# 📘 ReadFlow

**ReadFlow** is a local-first PDF reading Progressive Web App built for speed, focus, and zero distractions.

Stop switching between apps. Import your PDFs once. Read everything in one smooth flow.

---

## ⚡ Core Features

- 📂 Multi PDF import (batch selection)
- 🔄 Continuous reading flow (auto open next file)
- 🧠 Smart 3-file buffer system (previous, current, next)
- 📡 Offline-first with full PWA support
- 💾 IndexedDB storage (persistent local library)
- 📍 Reading progress tracking (resume anytime)
- 📊 Queue system with custom order and priority
- 🗂️ Reading plan (To Read, Reading, Completed)
- 👆 Gesture-based navigation for mobile
- 🎯 Clean, modern UI with proper spacing and readability
- 🔒 Password protection for local data security
- 🧹 Full delete system (no leftover data)

---

## 🚀 How It Works

- Import PDFs from your device using multi-select  
- Files are stored locally in your browser  
- Only **3 files** are active in memory at any time  
- The next file preloads automatically as you read  
- When you finish one file, the next opens instantly  

---

## ⚙️ Performance Design

Built for real mobile usage:

- ⚡ Lazy loading for PDFs and pages  
- 🚫 No mass rendering of large libraries  
- 📚 Supports **500 to 1000 PDFs** (library level)  
- 🧩 Controlled memory usage with smart loading  

---

## 📶 Offline Usage

This is a Progressive Web App:

- 📴 Works fully offline after first load  
- 📲 Installable on mobile and desktop  
- 🌐 No backend required  
- 🔐 No data leaves your device  

---

## 🔐 Security

- 🔑 Local password lock system  
- 🛡️ Protects your reading library  
- 📁 All data stored on-device only  

---

## 🧰 Tech Stack

- 🧱 HTML, CSS, JavaScript  
- 📄 pdf.js for PDF rendering  
- 💾 IndexedDB for storage  
- ⚙️ Service Worker for offline caching
